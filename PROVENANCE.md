repository became unavailable.

than importing a flattened copy.

Development after the recovered commit is intentionally separate in Git history. The first
continuation release:

- fixes the bootstrap wrapper on Windows when paths contain spaces;
- restores release, installer, security, and upgrade links to a maintained repository;
- adds Windows to continuous test coverage;
- refreshes skill metadata and public documentation;

Historical release notes and commits may contain links to the unavailable original repository.
They remain unchanged where doing so is important to the historical record.
